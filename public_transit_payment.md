## 📅 2일차 · 빈 시내 관광 (Vienna)

### 🚇 Wiener Linien (빈 시내 대중교통 — 메트로/트램/버스)

| 항목 | 내용 |
|:---|:---|
| **컨택리스 결제** | ⚠️ **탑승 시 직접 태그(tap-on/off) 불가** — 한국과 달리 단말기에 카드 태그하여 승차하는 방식 아님 |
| **티켓 구입 방법** | ① **WienMobil 앱** ★추천★ — Apple Pay / Google Pay 결제 지원<br>② 메트로 역 **자판기** — 컨택리스 카드/Apple Pay/Google Pay 가능<br>③ 역 내 매표소 · Tabak(담배가게) |
| **요금 (참고)** | 1회권 €2.40 / 24시간 €8 / 48시간 €14.10 / 72시간 €17.10 |
| **사용법** | 앱에서 구매 후 QR 활성화 or 자판기에서 종이 티켓 구매 → 메트로는 개찰구 없이 **자율 검표** (Kontrolle) |
| **💡 추천 전략** | **24시간권(€8)을 2일차 오후 ~15시에 활성화** → 3일차 오후 ~15시까지 유효 → 벨베데레 이동 + 저녁 귀환 + **3일차 아침 U3→Westbahnhof까지 커버!** |

> ⚠️ 빈은 **자율 검표 시스템** — 개찰구가 없어 무임승차 유혹이 있지만, 검표원이 자주 순찰하며 **벌금 €115**. 반드시 유효 티켓 소지!

### 📅 3일차 · 빈→잘츠부르크 이동 시 (U3 메트로)

일정에 `U3 메트로 → Wien Westbahnhof` 이동이 포함되어 있음 — ✅ **2일차 오후 활성화한 24시간권으로 커버됨** (별도 구매 불필요)

## 📅 3~5일차 · 잘츠부르크 (Salzburg)

### 🚌 잘츠부르크 대중교통 (Salzburg Verkehr)

| 항목 | 내용 |
|:---|:---|
| **🎉 Guest Mobility Ticket** | ✅ **숙소에서 무료 제공!** 숙박객 전원 대상, 체류 기간 동안 잘츠부르크 주(州) 전체 대중교통 무제한 이용 |
| **적용 기간** | 2025.05.01~2027.04.30 — **2026년 3월 체류 시 적용 ✅** |
| **포함 범위** | 시내버스, 트롤리버스, 지역버스, S-Bahn, 지역열차, 장거리열차 |
| **수령 방법** | 체크인 시 숙소에서 **PDF 또는 Wallet 패스**로 전달 |
| **비용** | 숙박비에 €0.50/박 모빌리티 요금 자동 포함 (15세 미만 무료) |
| **미포함** | 특별열차, 증기열차, 심야버스 |

> 💡 Villa Elisa 체크인 시 Guest Mobility Ticket을 꼭 요청하세요! 사전 온라인 체크인 가능한 경우 도착 전에 미리 받을 수도 있습니다. 이 티켓이 있으면 **시내버스 티켓을 별도 구매할 필요 없음**.

### 만약 Guest Mobility Ticket 없이 구매해야 할 경우

| 구매처 | 결제 방법 |
|:---|:---|
| **SalzburgMobil 앱** | 카드 결제 (컨택리스 포함) |
| **역/정류장 자판기** | 현금, 카드 (컨택리스 지원) |
| **버스 기사** | 현금 (약간 비쌈) |
| **TABAK/키오스크** | 현금/카드 — 사전 구매 시 할인 |

## 📅 6~10일차 · 프라하 (Prague)

### 🚇 PID 프라하 대중교통 (메트로/트램/버스)

| 항목 | 내용 |
|:---|:---|
| **컨택리스 결제** | ✅ **차량 내 단말기에서 직접 컨택리스 결제 가능!** 한국 교통카드와 유사한 경험 |
| **지원 결제수단** | Apple Pay / Google Pay / 컨택리스 VISA·Mastercard |
| **사용법** | 트램·버스: 컨택리스 결제 표시(💳)가 있는 문으로 탑승 → 단말기에 카드/폰 태그<br>메트로: 역 내 자판기 또는 앱으로 구매 |
| **📱 PID Lítačka 앱** ★강력 추천★ | Apple Pay / Google Pay 결제 지원, 72시간 교통권 등 모든 종류의 티켓 구매 가능, 영어 지원 |
| **자판기** | 메트로 역 내 — 현금(코루나) + 컨택리스 카드 |

> 💡 일정에 이미 **72시간 교통권 330 CZK** (3/12~3/14)이 계획되어 있습니다. **PID Lítačka 앱**에서 바로 구매하면 종이 티켓 없이 QR코드로 사용 가능! 3/15~16은 앱에서 개별 티켓(30분 30 CZK / 90분 40 CZK) 구매.

## 📅 8일차 · 드레스덴 당일치기 (Dresden)

### 🚌 DVB 드레스덴 대중교통

| 항목 | 내용 |
|:---|:---|
| **컨택리스 결제** | ⚠️ 탑승 시 직접 태그 불가 — 자판기/앱에서 구매 |
| **차내 자판기** | 트램/버스 내 설치 — 현금카드(EC카드) 및 스마트폰 NFC 결제 가능 |
| **📱 DVB mobil 앱** | 모바일 티켓 구매 가능 |
| **📱 FAIRTIQ 앱** | 체크인/아웃 방식 — 자동 최적 요금 계산 |
| **참고** | 4시간 20분 체류 중 **도보로 핵심 관광** (역↔구시가 왕복 도보 15분), 대중교통 이용이 없을 가능성 높음 |

> 📌 드레스덴 일정은 Dresden Hbf에서 도보 15분 거리의 구시가지를 걸어서 관광하는 계획이므로, **대중교통을 탈 일이 거의 없습니다.** 만약 필요하면 DVB mobil 앱 또는 차내 자판기 이용.
