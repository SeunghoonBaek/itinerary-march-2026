# 2026년 3월 유럽 여행 일정 프로젝트

## 개요
2026.03.06 ~ 03.16 (11일) 오스트리아-체코-독일 여행 일정을 관리하는 저장소.
경로: 빈(Vienna) → 잘츠부르크(Salzburg) → 프라하(Prague) → 드레스덴(당일) → 체스키 크룸로프(당일) → 프라하 출발

## 파일 구조

### `README.md` (주요 파일)
- 일정을 논의하고 업데이트하는 **유일한 원본 파일**
- 모든 일정 변경은 이 파일에서 반영한다

### `travel_map.html`
- Leaflet 기반 인터랙티브 지도
- 각 일정을 노드(장소)와 엣지(이동경로)로 시각화하고, 일자별 네비게이터로 탐색할 수 있다

## 식당 추천 규칙
1. Google Maps의 평점과 리뷰를 참고하여 추천한다
2. 운영시간을 반드시 조사한다 — 공식 사이트 또는 Google Maps 정보를 활용하고, 해당 일정 및 동선과 충돌이 없는지 확인한다
3. 식당의 대표 메뉴(유명한 것)와 추천 이유를 함께 안내한다
4. 호텔에 붙어있는 식당은 제외한다
5. 위치는 해당 일정의 동선에서 크게 벗어나지 않으면 된다

## 작업 규칙
1. 일정 변경 시 `README.md`를 수정한다
2. 일정 변경 후 `travel_map.html`에도 변경 내용을 동기화한다
